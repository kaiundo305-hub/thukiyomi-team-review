01 — Post Patterns

5つの投稿の型

この5パターンをローテーションするだけで、毎日の投稿に困らなくなります。

🌙
型① 今日の月宿（整えの声かけ）
推奨：週3〜4回 ／ X・Instagram両方

今日の宿をもとに「こんな日の整え方」を一言届ける。生活に密着した情報として認知が広がりやすい型。

今日は【宿名】の日🌙 【宿の特徴・エネルギー：1文】 こんな日は 【整えのアクション：具体的に1つ】 を試してみてください✨ 月の流れに合わせて整えると、 不思議と毎日が少しずつ変わってきます。 #月読みワンダーランド #月の整え #宿曜
▶ 実例
今日は「星宿（ほししゅく）」の日🌙 直感と感受性が高まる、繊細なエネルギーの日です。 こんな日は 好きな音楽を一曲だけ、目を閉じてゆっくり聴いてみてください✨ 月の流れに合わせて整えると、 不思議と毎日が少しずつ変わってきます。 #月読みワンダーランド #月の整え #宿曜
🌱
型② 整えの問いかけ（内省ワーク）
推奨：週1〜2回 ／ Instagram・X

読んだ人が「少し立ち止まれる」問いを届ける。保存率・コメント率が高い型。深い関係性が生まれる。

今日の問いかけ🌱 「【問いの内容：具体的で答えやすい1文】」 無理に答えなくていいです。 ただ、ふとした瞬間に思い出してみてください。 その「ふとした気づき」が、 自分を整えることの始まりです。 #月読みワンダーランド #自分を整える #今日の問い

✨
型③ 命宿の世界（宿の紹介）
推奨：週1回 ／ Instagram向き（保存される）

27宿のひとつを深掘りして紹介。「これ私だ！」と思った人がシェア・保存する型。27回分のネタになる。

【宿名】の方へ✨ あなたの本質は 「【命宿の核心：一言】」 【その宿の特徴・強み：2〜3文】 あなたにとっての開運習慣は、 【具体的な整えアクション】です。 自分の命宿を知ってみませんか？ プロフのリンクから無料で確認できます🌙 #月読みワンダーランド #命宿 #【宿名】 #宿曜占術
▶ 実例
「女宿（じょしゅく）」の方へ✨ あなたの本質は 「奥深い感性で、場の空気を整える人」 細やかな気遣いと美意識を持ち、 場をそっと整える力があります。 頑張りすぎてしまうのは、それだけ心が豊かな証拠。 あなたにとっての開運習慣は、 "丁寧な時間"を一日1つ作ることです。 自分の命宿を知ってみませんか？ プロフのリンクから無料で確認できます🌙 #月読みワンダーランド #命宿 #女宿 #宿曜占術
💛
型④ かのんエピソード（自己開示）
推奨：週1回 ／ X・Instagram両方

かのんさん自身の体験・気づきを語る。「人」として好きになってもらえる型。信頼関係の土台になる。

【昔の自分の状態：共感を呼ぶ一文】 【そのときの気持ち・悩み：具体的に】 でも、月読みと出会って気づいたこと。 「【気づきの核心：一文】」 【今の自分と変化：前向きに締める】 あなたにも、きっとその「気づき」の瞬間が来ます。 #月読みワンダーランド #自分を整える #かのん
▶ 実例
昔の私は、「修行しなければ」とずっと思っていました。 頑張ること・耐えること・克服することが 開運への道だと信じていたんです。 でも、月読みと出会って気づいたこと。 「整えることは、習慣でいい。」 今は毎日の小さな積み重ねを 月の流れに合わせて続けるだけ。 それだけで、人生が少しずつ変わっていきました。 あなたにも、きっとその「気づき」の瞬間が来ます。 #月読みワンダーランド #自分を整える #かのん
🏰
型⑤ ワンダーランドへの誘い（自然な誘導）
推奨：月2〜3回 ／ 売り込み感なしで

体験・診断・コンテンツへ誘導する型。「気になったら来てみて」のトーンで、絶対に押しつけない。

【読む人の状態・悩みへの共感：1〜2文】 月読みワンダーランドでは、 【何ができるか：体験として表現】 ことができます。 怖くないですよ。 占いではなく、自分を知るための「整えの旅」です🌙 気になった方は、プロフのリンクからどうぞ。 来るも来ないも、あなたのペースで大丈夫です。 #月読みワンダーランド #天命鑑定書 #無料診断
▶ 実例
「自分の本当の強みって何だろう？」 そう感じたことはありませんか？ 月読みワンダーランドでは、 生年月日から導いた命宿をもとに あなただけの天命鑑定書を受け取ることができます。 怖くないですよ。 占いではなく、自分を知るための「整えの旅」です🌙 気になった方は、プロフのリンクからどうぞ。 来るも来ないも、あなたのペースで大丈夫です。 #月読みワンダーランド #天命鑑定書 #無料診断

02 — Content Ideas

ネタ帳

型①〜⑤を埋めるためのネタアイデア集。思いついたらここに追記していく。

🌙 月宿ネタ（型①用） — 27宿分ストックできる

🌑 角宿：新しいことを始めるエネルギー。「今日、一つだけ新しい習慣を試してみて」
🌓 亢宿：整理と見直しの日。「引き出しを一つだけ片付けてみて」
🌕 氐宿：人とのご縁が深まる日。「大切な人に一言連絡してみて」
🌗 房宿：安定と守りの日。「ゆっくりお茶を一杯、自分のために」
🌑 心宿：感情が動く日。「感じたことを日記に書いてみて」
✨ 尾宿：直感が冴える日。「迷ったらまず動いてみて」

→ 残り21宿も同じ形で作れば、27日分の投稿ネタが完成

🌱 問いかけネタ（型②用）— テーマ別

💆 安心：「今週、誰かに"ありがとう"と言えましたか？」
🌿 習慣：「毎朝、自分のためにしていることはありますか？」
🌟 強み：「子どもの頃から変わらず好きなことは何ですか？」
🌙 夢：「10年後、どんな一日を過ごしていたいですか？」
🏡 居場所：「"ここにいていい"と感じる場所はどこですか？」
💛 自己肯定：「今の自分を、まず一つだけ褒めるとしたら？」
🌺 季節：「この季節に好きなものは何ですか？」
✨ 天命：「"使命"という言葉を聞いて、最初に浮かんだことは？」

💛 かのんエピソードネタ（型④用）

📖 言葉の変化：「修行」→「習慣」に変わった話
🌙 出会い：宿曜・月読みと出会ったきっかけ
😔 しんどかった時：一番つらかった時期と、何が助けになったか
✨ 気づきの瞬間：「あ、これが天命かも」と思った瞬間
🐱 にゃんてん誕生秘話：なぜ天使ネコのキャラクターにしたか
🌺 好きなもの：かのんさんの「整え習慣」ルーティン
💌 お客様からの声：受け取った言葉で嬉しかったもの
🏰 ワンダーランドの夢：10年後、どんな世界にしたいか

🏰 誘導ネタ（型⑤用）— 季節・タイミング別

🌑 新月：新しいことを始めたい方に天命鑑定書を
🌕 満月：今月の振り返りに7日間チャレンジを
🌸 年始・春：今年の自分の流れを知りたい方へ
🍂 秋・年末：来年に向けて自分を整える時期
🎂 誕生月：今年の天命を確認してみませんか
💆 しんどい時：落ち込んだ時こそワンダーランドへ

04 — Rules

SNS投稿のNG / OK

この世界観を守ることが、フォロワーを「仲間」にする。

🚫 絶対NG（禁止語・禁止表現）

✗ 「今すぐ」「残り○席」「急いで」
✗ 「危険な時期」「最悪の運気」
✗ 「このままではダメ」
✗ 「祟り」「霊」「呪い」「除霊」
✗ 「絶対にうまくいく」の断言
✗ 「乗り遅れると損」の煽り
✗ 「修行」（重さ・義務感が出る）
✗ 過剰な絵文字・全角大文字の強調

✅ 積極的に使うOK表現

✓ 「整える」「育てる」「感じる」
✓ 「習慣」（やさしく続けられる感）
✓ 「あなたのペースで」「来るも来ないも」
✓ 「少しだけ」「ひとつだけ」「ゆっくり」
✓ 「気になった方は」「もしよければ」
✓ 「ここにいていい」「大丈夫ですよ」
✓ 月・星・花・水のやさしいモチーフ
✓ 体験・感情・気づきの言葉


投稿前チェックリスト（3秒でOK）

□ 読んだ人が「急かされた」と感じないか？
□ 読んだ人が「不安になった」と感じないか？
□ 「売られた」と感じないか？
□ 50代女性が「品がある」と感じるか？
□ 「ここにいていい」という空気が出ているか？

5つ全部OKなら、投稿してください。

05 — Posting Rhythm

投稿リズムの目安

毎日完璧にやろうとすると続かない。まず「週4」から始める。

週4投稿スターターパック

月曜日：🌱 型② 問いかけ（週の始まりに内省）
水曜日：🌙 型① 月宿の声（今週の整え方）
金曜日：✨ 型③ 命宿の世界（週末に読まれやすい）
土日どちらか：💛 型④ かのんエピソード（週1回の自己開示）

型⑤（誘導）は月2〜3回を自然なタイミングで差し込む

投稿時間帯の目安（50代女性ターゲット）

朝 7〜8時：起床後の整え時間に読まれやすい
昼 12〜13時：ランチ・休憩中
夜 21〜22時：就寝前のリラックス時間

おすすめ：朝7時（起床後の静かな時間）


続けるための3原則

完璧より継続

短くてもいい。「今日の月宿」一言でも投稿した日が勝ち。

反応より世界観

いいねが少なくても世界観を守り続ける。ファンは静かに育つ。

型を信じる

毎回ゼロから考えない。5つの型をローテーションするだけでいい。
